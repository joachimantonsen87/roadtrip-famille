🍽️ Restos Dénichés — Pépites Locales

Pas de piège à touristes ! On mange comme les locaux, cuisine de qualité, prix doux, terrasses où les filles peuvent bouger.

🏔️

Aínsa — J3

  • 🍕 Pizzería La Tea — Plaza Mayor. Pizzas (sans gluten dispo) avec vue sur LA plus belle place d'Espagne. ~10-15€
  • 🥘 Bar Tapas L'Alfil — Croquettes de ouf dans une ruelle médiévale. Terrasse magique. ~8-12€
  • 🍳 Alberto Restaurante — Le favori des locaux. Tapas + braise. Terrasse. ~12-18€
  • ☕ La Koffeecleta — Café de spécialité dans le vieux village. Pause papa !
🌊

Cantabrie — J4-J6

  • 🦐 Asador El Águila (Noja) — Arroz negro + grillades. ~15-20€
  • 🍔 Amaya Bus (Noja) — Maison, abondant, prix mini. Le plan "on rentre affamés". ~8-10€
  • 🍦 El Ártico (Comillas) — LES glaces. Stop OBLIGATOIRE. Vote des filles !
  • 🌲 La Pérgola (Comillas) — Resto DANS un jardin de pins à la plage. Les filles jouent pendant que tu manges = THE plan solo dad
🏔️

Asturies — J7-J8

  • 🍎 Sidrería El Tarteru (Ribadesella) — LA sidrería. Cachopo + sidra. Ambiance 100% asturiana. ~12-18€
  • 🍽️ Mesón El Labrador (Ribadesella) — Menu complet ~10€ ! Cuisine simple, généreuse, familiale.
  • 🐟 Sidrería Carroceu — Anchoas con queso, chorizo a la sidra. ~10-15€
  • ⭐ Riu Calabrés (Llanes) — Fusion, 9.4/10. Pour se faire plaisir un soir. ~25€
🌊

Galice — J9-J12

  • 🐙 Pulpería Balneario (Baiona) — LE pulpo à la plancha. 14€. Stop rapide parfait.
  • 🍷 La Taberna del Pintor (Baiona) — Huevos rotos con pulpo y langostinos = plat signature. ~15-20€
  • 🦑 La Boquería (Baiona) — Tapas. Pulpo bipolar (à feira + brasa), chipirones. ~12-15€
  • 🥟 Queimada Gastrobar (Baiona) — Gyozas de langostinos, mini burgers. Parfait filles ! ~10-15€
  • 🫒 Olivando (Baiona) — Tostas créatives, sardines. Petit local adorable. ~10-15€
🦸‍♂️

Astuce Solo Dad au Resto

  • ✅ Terrasse toujours (les filles bougent, c'est OK dehors)
  • ✅ Raciones à partager = tout le monde pioche
  • ✅ Pan con tomate en entrée = les filles mangent pendant que tu choisis
  • ✅ Horaire espagnol: 14h déj, 21h dîner → filles mangent au camping 19h30, toi tu profites après

💡 Tips & Recommandations

Tout ce qu'il faut savoir pour un trip sans stress

⚠️

Distances à surveiller

  • J11 → J12 (Asturies → Baiona) : ~4-5h
  • J16 → J17 (Fisterra → Urbión) : ~7h !! Prévoir étape León/Burgos
  • Les autres trajets sont tous < 3h 👌
📅

Réservations prioritaires

  • 90 jours avant : Autorisation Cíes + Ferry
  • 60 jours avant : Campings (tous les 🔴)
  • Jour J : Billets PACO Corvera (9h en ligne !)
  • Été : Navette Covadonga obligatoire
🧳

Check-list Van + Enfants

  • Maillots de bain x3 par personne (ça sèche mal !)
  • Crème solaire SPF50 enfants
  • Chaussures eau + sandales rando
  • Jeux de carte, coloriages pour la route
  • Bouées / brassards plage
  • Lampe frontale (soirées camping)
  • Hamac si place dans le van
  • Trousse premier secours
🌡️

Météo par région

  • Huesca : Chaud 28-35°C, frais le soir en altitude
  • Cantabrie : 22-28°C, possible pluie — micro-climat
  • Asturies : 20-26°C, le « vert » = il pleut parfois !
  • Galice : 20-28°C, brises océaniques agréables
  • Urbión : 15-25°C, frais le soir — prévoir polaires
🍽️

Spécialités culinaires

  • Huesca : Ternasco (agneau), migas
  • Cantabrie : Sobao, quesada, anchois
  • Asturies : Fabada asturiana, sidra (cidre !)
  • Galice : Pulpo a feira, empanada, percebes
  • 🧒 Kids : Tortilla, croquetas, churros PARTOUT !
